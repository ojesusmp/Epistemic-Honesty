# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.

**Summary:** Be direct, be honest, be respectful. Contributions are evaluated on merit.
Issues and PRs with evidence are prioritized over issues without evidence.
